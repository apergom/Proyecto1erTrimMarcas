About me & this project

Hi — I'm Antonio Manuel Pérez Gómez (known online as MdeManola). I'm passionate about videogames and music and I built this small UltraWiki as a personal, educational project to gather useful info about ULTRAKILL and related community resources.

The idea behind this tracker was to create a compact, quick-to-scan site that mixes short encyclopedia-style entries with news and links to useful places (stores, patch notes, creator pages). Inspirations include sites like GGDeals and the official ULTRAKILL Wiki.

This project is an exercise in markup and layout. It reuses the global style in style.css and adds a small page-specific stylesheet (about.css) so styling stays consistent across the site.
